# 선택 정렬이란?
데이터가 무작위로 여러 개 있을 때, 처리되지 않은 데이터 중에서 가장 작은 데이터를 선택해 맨 앞에 있는 데이터와 바꾸는 과정을 반복하는 알고리즘을 선택 정렬이라 한다. **선택 정렬은 아이디어가 매우 간단하지만, 시간복잡도가 항상 O(N^2)로 다른 정렬 알고리즘과 비교했을 때 매우 비효율적이다.**
